1943: Type 4303. Verdens første kommercielle piezoelektriske accelerometer, som satte nye standarder for simpel måling af vibrationer.

1949: Brüel & Kjærs succes tager fart med introduktionen af Level Recorder Type 2301 – dette instrument og dets efterfølgere blev solgt i 1.000 eksemplarer om året i 25 år.

1958: Den første kondensatormikrofon bliver produceret. Den sætter standarden for industrien og er stadig førende inden for feltet.

1960: Type 2203, verdens første transistoriserede, håndholdte lydmåler.

1973: Brüel & Kjærs første hydrofon, Type 8101.

1977: Verdens første digitale filteranalysator, Type 2131, udviklet til at måle og vise oktav- og 1/3-oktav-spektre i realtid.

1981: Verdens første kommercielle system til analyse af lydintensitet, hvor såvel lydenergiens retning som dens størrelse kan måles.

1983: Verdens første modulære lydmåler, Type 2231. Anvendelsen blev øget betragteligt gennem optimerede applikationer.

1996: Verdens første Windows-baserede multi-analysator-system (PULSE), hvor man bruger PC'ens øgede processorkraft til at fremskynde målinger og analyser.

1997: Verdens første håndholdte lydintensitetssystem, Type 2260-E, hvor brugeren har mulighed for at identificere en lydkilde og bestemme dens effekt.

2004: Håndholdt analysator Type 2250 – verdens første lydmåler med farve-touch-screen.

2013: Sonoscout – verdens første tablet-baserede applikation til måling af lyd og vibration i køretøjer.
